# klasp

> Block AI coding agents (Claude Code, Cursor, Codex, Aider) on the same quality gates your humans hit at `git commit`.

This is the npm distribution of [klasp](https://github.com/klasp-dev/klasp). The
package is a thin shim that dispatches to a per-platform binary delivered via
optional dependencies — no install-time download, no postinstall script.

```sh
npm install -g @klasp-dev/klasp
klasp --version
```

For the full documentation, design doc, and roadmap, see
[github.com/klasp-dev/klasp](https://github.com/klasp-dev/klasp).
